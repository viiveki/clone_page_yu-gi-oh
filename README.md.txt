# Projeto Lista de Cartas Yu-Gi-Oh!

Projeto sobre a listagem de cartas inspiradas e que seguem o estilo das cartas da franquia Yu-Gi-Oh! As imagens dos personagem das cartas foram geradas por IA e o projeto foi desenvolvido acompanhando as lives da semana do programador realizadas pelo canal [Dev em Dobro](https://www.youtube.com/@DevemDobro).


## Sites utilizados

[CSS Scan](https://getcssscan.com/css-box-shadow-examples): modelos de sombras.

[favicon.cc](https://www.favicon.cc/?action=icon&file_id=795779): icon da carta para o site.
